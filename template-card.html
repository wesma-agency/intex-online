Описание

Каркасный бассейн Bestway 5612X. Чаша выполнена из высококачественного трехслойного ПВХ: два слоя плотного винила и между ними сетка из полиэстера для особой прочности. Цвет чаши бассейна снаружи синий. Каркас металлический, покрытый предотвращающим появление коррозии слоем пластика.


Установка каркасного бассейна Bestway 5612X максимально простая. Нужна лишь плоская горизонтальная поверхность. Нет необходимости в бетонировании площадки или подготовки ямы для последующего заглубления. Процесс сборки бассейна до наполнения водой занимает 30 минут.


В комплект поставки входят фильтр-насос производительностью 3028 л/ч для очистки и циркуляции воды, картридж, шланги и хомуты, необходимые для подключения к бассейну. Лестница для бассейна. Тент для бассейна
